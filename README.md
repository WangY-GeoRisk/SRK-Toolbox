# SRK Toolbox
Kriging with sparse representation (SRK) is an exact Kriging interpolator that directly reconstructs spatially varying geospatial data from limited measurements while quantifying the associated interpolation uncertainty.

This repository provides the **Kriging with sparse representation (SRK) Toolbox** for spatial interpolation and uncertainty quantification in ArcGIS Pro.

The toolbox is designed to generate spatially continuous prediction maps from limited measurement data and to quantify the corresponding interpolation uncertainty. The outputs include a predicted mean raster and a predicted standard deviation raster.

## Repository structure

```text
SRK-Toolbox/
│
├── SRK_toolbox/
│   └── Packaged SRK toolbox files
│
├── Example_input/
│   └── Example input shapefile
│
├── Example_output/
│   ├── Predicted mean raster
│   └── Predicted standard deviation raster
│
└── README.md
```

## Folder description

### `SRK_toolbox`

This folder contains the packaged SRK toolbox files that can be added to ArcGIS Pro.

### `Example_input`

This folder contains the example input point data used for interpolation.

The input data are provided as a shapefile. Please note that a shapefile consists of multiple associated files, such as:

```text
.shp
.shx
.dbf
.prj
.cpg
```

All associated files should be kept in the same folder.

### `Example_output`

This folder contains the example interpolation results generated by the SRK toolbox, including:

- predicted mean raster
- predicted standard deviation raster

Since the output rasters are stored in Esri GRID format, all files within each raster folder are required and should not be renamed individually.
These output files can be directly opened and visualized in ArcGIS Pro.

## Usage

1. Open ArcGIS Pro.
2. Go to `Insert` → `Toolbox` → `Add Toolbox`.
3. Select the SRK toolbox from the `SRK_toolbox` folder.
4. Open the SRK toolbox in the Geoprocessing pane.
5. Set the input point feature from the `Example_input` folder.
6. Select the corresponding value field for interpolation.
7. Specify the output paths for the mean raster and standard deviation raster.
8. Run the toolbox.

## Outputs

The SRK toolbox produces two main raster outputs:

- **Mean raster**: the predicted spatial distribution of the target variable.
- **Standard deviation raster**: the spatial distribution of interpolation uncertainty.
